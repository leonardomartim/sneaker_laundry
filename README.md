O **Sneaker Laundry** é uma plataforma moderna desenvolvida para capturar solicitações de orçamento de clientes de forma automatizada. O sistema valida dados cadastrais, localiza endereços dinamicamente via API de CEP, persiste os dados em um banco relacional estável e dispara notificações por e-mail.

## 🚀 Tech Stack

* 🦀 **Rust:** Backend focado em altíssima performance, confiabilidade e segurança de memória.
* ⚡ **Vite + React:** Build otimizado, tipagem via TypeScript.
* 🎨 **Tailwind CSS v4:** Estilização utilitária.
* 🐘 **PostgreSQL:** Persistência relacional estruturada.
* 🐳 **Docker:** Contêinerização de ambientes (Dev/Prod).
* 📍 **API ViaCEP:** Integração assíncrona para auto-preenchimento de endereços e validação rigorosa (bloqueio de CEPs inexistentes).

* **Clean Architecture:** Estrutura modular.
* **Security-First:** Proteção de credenciais.
